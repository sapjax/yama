# Spec Delta: Settings - Custom Panel Width

## ADDED Requirements

### Requirement: Custom Panel Width
The application MUST allow users to configure the width of the dictionary panel.

#### Scenario: User changes panel width
1. User opens the Options page.
2. User navigates to the "Miscellaneous" tab.
3. User enters a valid number (e.g., 700) into the "Panel Width" input.
4. The setting is saved.
5. When the dictionary panel is opened, it renders with the specified width (700px).
